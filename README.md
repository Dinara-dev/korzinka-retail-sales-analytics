


# 🛒 Korzinka Retail Sales Analytics Dashboard

## 📊 Overview
Power BI dashboard analyzing retail sales performance,
product rankings, and regional revenue distribution
for Korzinka — one of Uzbekistan's largest supermarket chains.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Excel / CSV data source

## 📌 Key Metrics
- Total Sales: 17.34 MLN so'm
- Total Quantity: 1,358
- Average Price: 12.75 K so'm
- Top Product: Cola (2.3 MLN so'm)
- Top Region: Farg'ona (3.7 MLN so'm)

## 📈 Dashboard Features
- Top purchased products ranking
- Regional sales distribution by city
- Product category breakdown (Oziq-ovqat vs Ichimlik)
- Low stock products list (Noyob mahsulotlar)
- Overall product sales trend chart

## 🗂️ Categories
- 🥗 Oziq-ovqat (Food): 60.82%
- 🥤 Ichimlik (Beverages): 39.18%
